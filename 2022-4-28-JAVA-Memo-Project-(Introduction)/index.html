Java Memo

I like to write memos on my computer. However, I did not find any memo software that can be locked. By happened, these days, I have received a project related to java. One idea comes to my mind: why don’t I use java classes to create my memo software? If I cannot find software satisfied my demand, why don’t I create it by myself. Thus, my finial java project is JAVA Memo.
